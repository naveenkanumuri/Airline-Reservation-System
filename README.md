# Airline-Reservation-System
**How to Run this system?**
1. Clone this repository in your local drive.
2. Run the command in terminal **python manage.py runserver**
3. After this the system will start working for the user.
4. But if you want to run it as admin then you have to do certain steps.
5. In the http address write the address like **127.0.0.1:8000/admin/**
6. After this admin panel of Django will be opened.
7. There you can see all the details of system.
8. There you can add and delete the flights.
